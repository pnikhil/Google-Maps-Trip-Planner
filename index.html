Road Trip Planner

A Google Maps API implented product, this Road Trip Planner application helps you find the detailed route for the given destination including the total distance and approximate time needed to cover the journey. In case if you are planning a journey that involves multiple locations, you can enter them in any order and the application will help you in choosing the best possible routes so as to cover the whole trip in the minimal span of time.
